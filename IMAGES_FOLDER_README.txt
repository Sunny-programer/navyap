PLACE STUDENT PHOTOS HERE
==========================

Naming Convention:
- Use the roll number as the filename
- Supported formats: .jpg, .png, .webp

Examples:
- images/101.jpg (for student with roll number 101)
- images/102.png (for student with roll number 102)  
- images/103.webp (for student with roll number 103)

The application will automatically find and display these images when a student views their report.
